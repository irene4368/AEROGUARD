# AeroGuard ✈️

AeroGuard is an aviation risk analysis system that evaluates flight safety
based on aircraft maintenance, crew fatigue, and weather conditions.

## Features

- Aircraft database
- Crew fatigue analysis
- Weather risk engine
- Flight simulation
- Safety dashboard
- Risk score visualization

## Tech Stack

Backend: Flask
Database: SQLite
Frontend: HTML, CSS, JavaScript
Charts: Chart.js

## How It Works

1. Add aircraft data
2. Add crew data
3. Simulate flight conditions
4. Risk engine calculates safety score
5. Dashboard displays flight risk analytics
